**Advocate Case Assignment System**
This is a small Java console project where advocates can be assigned different legal cases. The program uses basic OOP and Collections like Map and List.
The project includes two classes:

AdvocateCaseService – contains the logic for adding advocates, assigning cases, updating status, and showing case details.
AdvocateCaseController – runs the program and demonstrates how the service works.

Features

Add new advocates
Assign cases using case ID + title
Prevent duplicate case IDs
Update case status (OPEN, IN‑PROGRESS, CLOSED)
Display cases of one advocate
Display all advocates and their cases


This project is mainly for practicing Collections, OOP, and simple conditional checks in Java.
